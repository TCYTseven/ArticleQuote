# ArticleQuote v1.0

ArticleQuote is a lightweight Chrome extension that turns any highlighted text on the web into a clean, shareable quote image. It’s meant for readers, students, researchers, and journalists who want a quick way to capture key lines from articles without dealing with screenshots or manual design work.

Highlight text, right-click, choose **“ArticleQuote”**, and the editor opens with all the essentials to style and export your image.

## Features
- Right-click → **ArticleQuote** on any selected text  
- Clean, modern output with centered quote text  
- Optional citation (URL or MLA-style, pulled from the page)  
- Background library with solid colors and gradients  
- Adjustable card size, border radius, shadows, typography, and spacing  
- Word-level highlight controls with customizable colors  
- Optional date/timestamp  
- Export as PNG or copy the image to clipboard  
- Minimal, straightforward UI designed for speed and clarity  

## Installation (Developer Mode)
1. Open Chrome and go to `chrome://extensions/`  
2. Enable **Developer mode**  
3. Click **Load unpacked**  
4. Select the extracted extension folder
